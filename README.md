# Installation Steps



## Using npm

Run commands

1) ```npm install```


2) ```npm run dev```


## Or using yarn

Run commands 

1) ```npm install --global yarn```

2) ```yarn install```

3) ```yarn run dev```

### This project is Build Using

1) ReactJS
2) Redux
3) firebase 
4) NextJs
5) Tailwind CSS
